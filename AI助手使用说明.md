# AI数据助手使用说明

## 功能概述

仪表盘现在集成了AI对话助手，可以帮助你快速查询和分析数据，无需手动查看各个图表。

## ⚙️ 配置说明

### 团队统一配置

✅ AI助手已由团队统一配置，**无需个人配置API Key**，可直接使用。

### 技术实现

- API Key 安全存储在 Cloudflare Worker 环境变量中
- 前端通过 Worker 代理调用硅基流动API
- 所有团队成员共享同一个 API 配额

## 如何使用

### 1. 打开聊天窗口

- 点击页面右下角的**蓝色圆形聊天按钮**（带对话框图标）
- 聊天窗口会从右下角弹出

### 2. 提问方式

AI助手可以回答以下类型的问题：

#### 📊 数据查询
- "最近7天有多少用户？"
- "进入问诊率是多少？"
- "完成率如何？"
- "平台总共有多少用户？"

#### 📈 趋势分析
- "用户增长趋势如何？"
- "留存率表现怎么样？"
- "和上周相比有什么变化？"

#### 🏥 科室数据
- "哪个科室最受欢迎？"
- "有多少人选择了内科？"
- "Top 5热门科室是哪些？"

#### 🔍 用户行为
- "有多少首次用户？"
- "回访用户占比多少？"
- "诊断行为率是多少？"

### 3. AI助手特点

✅ **实时数据**：基于当前仪表盘显示的数据范围进行分析
✅ **上下文理解**：可以进行多轮对话，理解前后文关系
✅ **专业分析**：提供简洁、专业的数据解读
✅ **快速响应**：使用硅基流动的Qwen2.5-7B模型，响应速度快

### 4. 示例对话

**用户**：最近7天有多少用户？
**AI**：根据当前数据（2026-01-19 至 2026-01-26），最近7天共有234位用户访问了平台。其中首次用户156人，回访用户78人。

**用户**：进入问诊率怎么样？
**AI**：当前进入问诊率为68.5%，表示100个用户中有约69人进入了问诊环节。这个转化率处于较好水平。

**用户**：哪个科室最受欢迎？
**AI**：根据统计，最受欢迎的前三个科室分别是：内科（89次）、外科（67次）、儿科（45次）。内科明显领先。

## 技术说明

- **AI模型**：硅基流动 Qwen/Qwen2.5-7B-Instruct（免费模型）
- **部署方式**：Cloudflare Worker 代理
- **API Key 管理**：由团队通过 Cloudflare Worker 环境变量统一管理，不暴露于前端
- **数据来源**：当前仪表盘显示的数据范围
- **更新频率**：每天凌晨1点（UTC）自动更新数据
- **隐私保护**：对话不会存储到服务器，仅在当前会话有效

## 注意事项

⚠️ AI助手的回答基于当前筛选的数据范围，如果你更改了日期范围或筛选条件，请重新提问以获取准确数据。

⚠️ AI助手使用的是免费API，如遇到响应延迟或限流，请稍后再试。

⚠️ 建议提问时尽量具体明确，这样可以获得更准确的答案。
