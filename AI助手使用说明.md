# AI数据助手使用说明

## 功能概述

仪表盘现在集成了AI对话助手，可以帮助你快速查询和分析数据，无需手动查看各个图表。

## ⚙️ 首次使用配置

### 1. 获取免费 API Key

AI助手使用硅基流动的免费模型。首次使用需要配置您自己的 API Key：

1. 访问 **https://siliconflow.cn**
2. 注册/登录账号（支持微信/GitHub登录）
3. 进入控制台，创建 API Key
4. 复制您的 API Key

### 2. 配置 API Key

- 首次点击 AI 助手按钮时，系统会自动弹窗要求输入 API Key
- API Key 会保存在浏览器本地（localStorage），不会上传到服务器
- 如需更换 API Key，清除浏览器数据后重新配置即可

### 3. 免费额度说明

- 硅基流动提供每月免费额度
- 如提示"余额不足"，请访问 https://siliconflow.cn 充值或等待下月额度刷新

## 如何使用

### 1. 打开聊天窗口

- 点击页面右下角的**蓝色圆形聊天按钮**（带对话框图标）
- 聊天窗口会从右下角弹出

### 2. 提问方式

AI助手可以回答以下类型的问题：

#### 📊 数据查询
- "最近7天有多少用户？"
- "进入问诊率是多少？"
- "完成率如何？"
- "平台总共有多少用户？"

#### 📈 趋势分析
- "用户增长趋势如何？"
- "留存率表现怎么样？"
- "和上周相比有什么变化？"

#### 🏥 科室数据
- "哪个科室最受欢迎？"
- "有多少人选择了内科？"
- "Top 5热门科室是哪些？"

#### 🔍 用户行为
- "有多少首次用户？"
- "回访用户占比多少？"
- "诊断行为率是多少？"

### 3. AI助手特点

✅ **实时数据**：基于当前仪表盘显示的数据范围进行分析
✅ **上下文理解**：可以进行多轮对话，理解前后文关系
✅ **专业分析**：提供简洁、专业的数据解读
✅ **快速响应**：使用硅基流动的Qwen2.5-7B模型，响应速度快

### 4. 示例对话

**用户**：最近7天有多少用户？
**AI**：根据当前数据（2026-01-19 至 2026-01-26），最近7天共有234位用户访问了平台。其中首次用户156人，回访用户78人。

**用户**：进入问诊率怎么样？
**AI**：当前进入问诊率为68.5%，表示100个用户中有约69人进入了问诊环节。这个转化率处于较好水平。

**用户**：哪个科室最受欢迎？
**AI**：根据统计，最受欢迎的前三个科室分别是：内科（89次）、外科（67次）、儿科（45次）。内科明显领先。

## 技术说明

- **AI模型**：硅基流动 Qwen/Qwen2.5-7B-Instruct（免费模型）
- **数据来源**：当前仪表盘显示的数据范围
- **更新频率**：每天凌晨1点（UTC）自动更新数据
- **隐私保护**：对话不会存储到服务器，仅在当前会话有效

## 注意事项

⚠️ AI助手的回答基于当前筛选的数据范围，如果你更改了日期范围或筛选条件，请重新提问以获取准确数据。

⚠️ AI助手使用的是免费API，如遇到响应延迟或限流，请稍后再试。

⚠️ 建议提问时尽量具体明确，这样可以获得更准确的答案。
